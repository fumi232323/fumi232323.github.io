# VAGRANT-END: 501 e9fe3cbf-63c6-48a1-beca-2702bc2527e4

  • 今の /etc/exports

    FumienoMacBook-Pro:ansible fumi23$ cat /etc/exports
# VAGRANT-BEGIN: 501 e9fe3cbf-63c6-48a1-beca-2702bc2527e4
